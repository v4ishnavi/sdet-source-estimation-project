# SDET Source Estimation Project

This project implements source detection and estimation algorithms using MATLAB. The codebase includes:

- **2D Source Localization** (`fig72d.m`) - Estimates emitter location using a 2D sensor array
- **3D Source Localization** (`fig73d.m`) - Extends localization to 3D space with multiple sensors
- **Signal Processing** (`tryfig8.m`) - Implements chirp signal processing and time-frequency analysis

The algorithms simulate radar/RF signal detection scenarios with multiple sensors positioned at known coordinates to triangulate and estimate the location of signal emitters.

## Files
- `fig72d.m`, `fig73d.m`, `tryfig8.m` - Main MATLAB simulation scripts
- `*.jpg` - Generated visualization outputs
- `sdet_report.pdf`, `sdet_ppt.pdf` - Project documentation
